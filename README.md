## 🎯 What is MindShant?

MindShant is a launch-ready **wellness + e-commerce + education platform** combining:

- 🛍️ **E-commerce Store** — Mindful DIY craft kits, journals & activity boxes
- 🌿 **Wellness Platform** — Free guides, mindfulness resources & screen-time plans
- 📝 **Educational Content Hub** — SEO-optimised blog on digital detox & mindfulness
- 👨‍👩‍👧 **Community Platform** *(roadmap)* — Workshops, coaching, memberships

## 🚀 Local Development (v1 Static)

```bash
# Clone the repository
git clone https://github.com/Jyoti2823/mindful-crafts.git
cd mindful-crafts

# Open in browser (no build step required)
open index.html
# or serve with any local server:
npx serve .
# or
python3 -m http.server 3000
```
